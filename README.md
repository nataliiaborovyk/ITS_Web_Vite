# ITS_Web_Vite – Esercizi Web con Vite  (ITS ICT Academy)

Questo repository contiene esercizi relativi allo sviluppo web con **Vite**, realizzati durante il corso di sviluppo web presso ITS ICT Academy.

Gli esercizi includono:

- configurazione di progetti con Vite
- struttura moderna di progetti frontend
- piccoli esperimenti e applicazioni web

Il repository rappresenta una fase intermedia del percorso di apprendimento dello sviluppo web.

Questo repository fa parte del mio percorso di apprendimento durante il programma Data Analyst presso ITS ICT Academy.
